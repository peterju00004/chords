# Chords

### Installation
1. Clone repository.
    ```bash
    git clone https://github.com/peterju00004/chords.git
    ```
2. Switch to the project root directory.
3. Install dependency.
    ```bash
    npm i
    ```
4. To run: `npm run dev`.
